# 🏛️ Sovereign Proverbs: Anchor Protocol
### *Part of the Sovereign Recovery™ Suite*

Sovereign Proverbs is a high-performance "Anchor" application designed for daily mental grounding and spiritual discipline. Unlike standard bible apps, this is a **Sovereign PWA (Progressive Web App)**—it is designed to function entirely offline, independent of the cloud, and free from the distractions of the modern web.

## 🛡️ The Sovereign Philosophy
This app is built on the **Alliance Guardian** design system. It follows the "Operator" mindset:
- **Minimalism:** No social feeds, no ads, no trackers.
- **Sovereignty:** Your data stays on your hardware.
- **Discipline:** A structured 31-day cycle that resets every month.

## ⚡ Key Features
- **The Survival Engine:** Integrated Service Worker (`sw.js`) allows the app to load instantly without an internet connection once installed.
- **The Voice Engine:** A calibrated text-to-speech interface for hands-free meditation during training or travel.
- **Pattern Interrupts:** Randomly triggered "Agape" mindset quotes to break negative thought loops.
- **Native Experience:** Fully responsive UI that installs directly to the mobile home screen as a standalone application.

## 🛠️ Technical Stack
- **Architecture:** Vanilla JavaScript / HTML5 / CSS3 (Zero external dependencies).
- **Audio:** Web Speech API (Native browser synthesis).
- **Storage:** Offline Caching via Cache API.
- **UI:** Alliance Guardian Crimson-Grey Schema.

## 📁 File Structure
- `index.html`: The core interface and PWA registration.
- `proverbs.js`: The "Wisdom Index" containing the full 31-chapter dataset.
- `styles.css`: The Alliance Guardian design system.
- `sw.js`: The Survival Engine (Offline logic).
- `manifest.json`: Mobile identity and home-screen configuration.
